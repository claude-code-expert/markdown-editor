## 계약 규칙

| # | 규칙 | 검증 |
|---|------|------|
| D1 | 저장된 전 문서를 리스트로 표시(SC-001·FR-001) | 전수 표시 |
| D2 | 각 항목: 제목·소속 폴더·수정 시각 등 식별 정보(FR-002) | 항목 정보 |
| D3 | 항목 선택 → `/editor/[id]` 이동(FR-003) | 네비게이션 |
| D4 | 문서 0 → 빈 상태 안내 + 시작 수단(US1 빈상태) | 빈 상태 |
| D5 | 리스트·항목 한글 aria + 키보드 접근(FR-012·SC-008) | 접근성 |
| D6 | anti-slop(M1 토큰, 그라데이션·글로우 없음) | 시각 |
