# Contract: 저장 시 폴더 선택 (`FolderSelect.tsx` + workspace)

## 동작

- 상단에 폴더 드롭다운. 활성 문서의 현재 폴더 표시. 선택 변경 → `moveDocument(activeDocId, folderId)`.
- 저장 시 선택 가능한 폴더가 없으면 차단 + 안내. 중복 제목은 `uniqueTitle` 접미사.

## 계약 규칙

| # | 규칙 | 검증 |
|---|------|------|
| F1 | 폴더 드롭다운이 전 폴더를 선택지로 제공(FR-006) | 목록 |
| F2 | 선택 → 활성 문서 폴더 변경·반영(`moveDocument`) | 이동 |
| F3 | 선택 가능한 폴더 없음 → 저장 차단 + 안내(FR-007·SC-004) | 차단 |
| F4 | 폴더 지정 후 저장 → 해당 폴더에 저장·성공 피드백(FR-008·SC-005) | 저장 |
| F5 | 동일 폴더 중복 제목 → `uniqueTitle` 접미사(`-1`,`-2`)(FR-009) | 중복 처리 |
| F6 | 셀렉터 한글 aria + 키보드(FR-012) | 접근성 |

## 순수 함수 (테스트 가능)

```typescript
uniqueTitle(existing: string[], title: string): string
```

| # | 규칙 | 검증 |
|---|------|------|
| U1 | 중복 없으면 원본 반환 | `["a"],"b"→"b"` |
| U2 | 중복 1개 → `-1` | `["메모"],"메모"→"메모-1"` |
| U3 | `-1`도 있으면 `-2` | `["메모","메모-1"],"메모"→"메모-2"` |
